# Image Fusion Method Based on Convolutional Sparse Representation and Fractional Order Differentiation in Wavelet Domain
-  Special thanks to **Ingrid Daubechies** (2023 Wolf Prize Laureate of Math)
    - First **Female** Wolf Prize Winner
    - Best known for her work with **wavelet theory** in image processing.
- Process
    - Discrete wavelet transform
    - Low freqency subband →  ****Fractional Calculus → fusion****
    - High freqency subband →  ****Convolutional Sparse Representation → fusion****
    - Inverse Discrete wavelet transform
